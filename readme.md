#Daze landing

##run:
```sh
npm i
```
### then create ./app/config/parameters.json with:
```json
{
  "mailUser": "some_mail@mail.ru",
  "mailPassword": "your_password@mail.ru"
}
```
### sync database:
```sh
npm run sync
```

### run server:
```sh
npm run serve
```
### see logs:
```sh
npm run logs
```
### stop server:
```sh
npm run unserve
```
